담당 업무

1
학교신청 선택화면
설계 / 프론트엔드

장학금 신청 시 학교 검색 및 선택 UI 설계. 학교명 자동완성, 학과 선택 연동, 모바일 터치 최적화 구현. 기존 PC 전용 화면을 멀티 디바이스형으로 전환.

2
H-보증제도 선택 화면
설계 / 프론트엔드

보증인 제도(인적보증/기관보증) 선택 플로우 설계. 각 제도별 필요 서류 안내, 조건 분기 처리. 복잡한 선택지를 단계별 위자드 형태로 UX 개선.

3
'K-특기정보'·'L-가정사정정보'·'긴급·응급채용정보' 입력 화면
설계 / 프론트엔드

다수의 입력 필드가 있는 복잡한 폼을 모바일에서도 사용하기 쉽게 개선. 입력 검증, 실시간 피드백, 임시저장 기능, 이전 단계 데이터 유지 처리.

4
M-장학금 입금 계좌 정보
설계 / 프론트엔드

은행 선택과 우체국은행 선택 시 입력 폼이 다른 분기 처리 구현. 계좌번호 형식 검증, 확인 화면에서 마스킹 처리. 입력 오류 시 친절한 에러 메시지 제공.

은행 선택 케이스 우체국은행 선택 케이스 계좌 정보 확인 화면
